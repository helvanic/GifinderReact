This website was developped by Victor Dupuy.

It's a little project I developped on my free time. The technologies I used are HTML5, CSS (via SASS), Javascript ES6 (Especially the class system, with Gulp-babel to make it readable on the browser), VelocityJS on the front-end. For the backend, I used NodeJS, ExpressJS, MongoDB (via MongoLabs on distant and MongoDB for testing). The website is deployed on Heroku. For automating build, I used Gulp.
